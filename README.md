🍽️ Food Recommendation System

 📌 Problem
Users struggle to choose food due to too many options.

 🎯 Solution
Built a hybrid recommendation system using:
- Content-based filtering
- Collaborative filtering

 ⚙️ Tech Stack
Python, Pandas, Scikit-learn

 📊 Results
- Precision@5: 0.78
- RMSE: 1.12

🚀 Features
- Personalized recommendations
- Similar food suggestions.
