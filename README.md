# customer-churn-analysis-excel
Identify key factors driving customer churn in a telecom company and provide actionable business insights.
📌 Project Overview

This project analyzes telecom customer data to identify churn patterns and provide actionable business insights.

🎯 Business Objective

Reduce customer churn by identifying key drivers such as contract type, tenure, and payment method.

🛠 Tools Used

Microsoft Excel

Pivot Tables

Slicers

Data Cleaning Techniques

KPI Dashboard Design

📊 Key Insights

27% overall churn rate

Customers with tenure <12 months show highest churn (48%)

Month-to-month contracts drive majority of churn

Electronic check payment users show higher churn tendency

📈 Dashboard Features

Interactive filters (Contract, Gender, Internet Service)

KPI section

Churn segmentation analysis

Business recommendations

💡 Business Recommendations

Improve onboarding for new customers

Incentivize long-term contracts

Offer discounts for high-risk churn segments
